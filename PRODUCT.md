# Product

## Register

product

## Users

面向公开使用的中文算法学习者，包括刚开始系统刷题的学生、准备技术面试的开发者，以及希望同时练习 Python 和 C++ 的用户。用户通常处于专注练习场景，需要快速定位题目、独立编码、运行测试、理解标准解法并安排复习。

## Product Purpose

把公开的 LeetCode 题目目录、原创中文题意摘要、分类学习路径、Python/C++ 参考实现、复杂度分析和浏览器内练习工作台整合在一个网站中。成功意味着用户可以从分类选题到编码、运行、复盘与继续下一题形成完整闭环，同时始终能回到 LeetCode 官方页面核对原题与提交。

## Brand Personality

专注、可信、高效。界面应具有成熟代码工具的秩序感，内容表达直接清晰，在长时间刷题时保持安静，不以装饰争夺注意力。

## Anti-references

不做营销落地页式的大标题和卡片堆叠，不使用花哨渐变、玻璃拟态或无意义动效，不复制 LeetCode 的品牌资产或制造官方从属关系，也不把完整受版权保护的题面作为自有内容批量转载。

## Design Principles

1. 练习优先：进入页面后立即能选题、编码和运行。
2. 熟悉但独立：保留成熟刷题平台的工作流，不进行像素级仿制。
3. 解释可验证：每个参考解法都包含核心推导、复杂度与测试用例。
4. 分类即路径：标签不仅用于筛选，还要帮助用户形成循序渐进的学习路线。
5. 渐进揭示答案：默认鼓励独立思考，再按需查看提示、思路与标准实现。

## Accessibility & Inclusion

以 WCAG 2.2 AA 为基线，支持键盘操作、清晰焦点状态、足够对比度、色觉友好的难度标识、减少动态效果偏好，以及桌面与移动端的结构化响应布局。
