## 解法指南

每个 C++ 文件应遵循以下格式：

```cpp
/*
题号: XXXX
题名: Problem Name
难度: Easy/Medium/Hard

时间复杂度: O(n)
空间复杂度: O(n)
*/

#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    // 实现代码
};
```

## 编译和运行

```bash
# 编译
g++ -std=c++17 -o solution solution.cpp

# 运行
./solution
```

## 学习路径

### 第一阶段：基础数据结构
1. 数组和字符串
2. 链表
3. 栈和队列
4. 哈希表（unordered_map）
5. 树和图

### 第二阶段：基础算法
1. 排序算法
2. 查找算法
3. 双指针
4. 滑动窗口
5. 递归和回溯

### 第三阶段：进阶算法
1. 动态规划
2. 图论算法
3. 贪心算法
4. 分治算法
5. 位操作
